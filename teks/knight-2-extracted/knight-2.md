 IMPLEMENTATION GUIDE FOR SUBSTITUTES
DAILY PREPARATION (15 minutes): 1. Read lesson plan script in main curriculum 2. Review presentation slides 3. Print student handouts (one per student + extras) 4. Gather materials needed 5. Set up classroom/technology
DURING CLASS (75 minutes): Minutes 0-15: I DO (Teacher presents using script) Minutes 15-30: YOU DO (Students work independently) Minutes 30-45: WE DO (Facilitated discussion/activity) Minutes 45-75: WORK TIME (Projects/assignments)
CLASSROOM MANAGEMENT TIPS: Use timer to keep on schedule Circulate during work time

Call on non-volunteers for equity Follow script closely if unsure Collect work if indicated in plan Note any issues for regular teacher
IF YOU NEED HELP: Scripts provided - read directly if needed Modify timing if students need more/less time Skip optional extensions if short on time Contact department chair for emergencies Answer keys available for worksheets
