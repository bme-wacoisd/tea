# Group Discussion: Copyright and Fair Use

## Prompt 1: "I Found It on Google"
Why isn't "I found it on Google" a valid defense for using an image in your classroom presentation? What does Google Image Search actually show you?

## Prompt 2: Fair Use Factors
What are the four factors courts consider when determining fair use? Which of these factors might favor a teacher using copyrighted material?

## Prompt 3: Finding Legal Resources
How would you find a legally usable image of the Texas Capitol for a social studies lesson? Walk through your search strategy.

## Prompt 4: The "It's Educational" Myth
A colleague says "It's for education, so copyright doesn't apply." How would you respond? What's wrong with this thinking?

## Prompt 5: Modeling Attribution
Why should teachers model proper attribution even in cases where it might technically be legal not to? What message does this send to students?

## Prompt 6: AI-Generated Content
How might AI-generated images and text change how we think about copyright and original work? What new questions does this technology raise?
